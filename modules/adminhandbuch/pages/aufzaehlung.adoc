Im Bereich Aufzählung in der Administration werden für die 

- Aktivitäten,
- Dokumentationskategorien und
- Aufgaben-Prioritäten

die Werte festgelegt. Dazu steht folgende Maske zur Verfügung.

image:adminhandbuch/aufzaehlungen_liste.png[Aufzählungen]

Für die drei Aufzählungen sind jeweils die Listen mit den gepflegten Werten dargestellt.

== Wert hinzufügen

Zwischen der Überschrift der einzelnen Aufzählungen und den dazugehörigen Tabelle ist jeweils ein Button image:adminhandbuch/button_neuer-wert.png[] platziert, über den zu der darunterliegenden Tabelle Werte hinzugefügt werden können. Mit Klick auf den Button erscheint folgende Maske:

image:adminhandbuch/aufzaehlungen_neuer-wert.png[Aufzählungen neuer Wert]

Hier wird der Name für den neuen Wert vergeben. Weiter kann angegeben werden, ob der Wert Aktiv ist und damit für die Nutzung freigegeben ist. Weiter kann ein der Wert in der Liste als Standardeinstellung festgelegt werden. Wird das Häkchen gesetzt, so wird dieser Wert der Standardwert. Der bisherige Standardwert wird mit dem Setzen entzogen.

Mit Klick auf den Button image:adminhandbuch/button_anlegen.png[Wert anlegen] wird der neue Wert der Liste hinzugefügt.

== Wert bearbeiten

Klickt man auf den Namen des zu bearbeitenden Werts, so gelangt man in die Bearbeitungsmaske des Werts.

image:adminhandbuch/aufzaehlungen_wert-bearbeiten.png[Wert bearbeiten]

In dieser Maske kann der Name verändert werden, der Wert Aktiv/Passiv geschaltet werden oder auch der Standardwert verändert werden.

Mit dem image:adminhandbuch/button_speichern.png[Wert speichern] werden die Änderungen übernommen.

== Reihenfolge bearbeiten

Die Reihenfolge in den Aufzählungen Listen bestimmt, in welcher Reihenfolge die Werte in der Anwendung gelistet werden. Durch Klick und Halten der linken Maustaste auf das Icon image:adminhandbuch/button_verschieben.png[Verschieben] kann die Zeile mit dem Wert in der Liste verschoben werden.

image:adminhandbuch/aufzaehlungen_reihenfolge.png[Reihenfolge Aufzählungen]

An der Position, an der die linke Maustaste gelöst wird, wird der Wert platziert.

== Wert löschen

Über den Button image:adminhandbuch/button_loeschen.png[Wert löschen] kann der Wert gelöscht werden. Wenn der Wert in der Anwendung bereits vergeben wurde, erscheint folgende Maske.

image:adminhandbuch/aufzaehlungen_loeschen_zuweisung.png[]

In dieser Maske muss angegeben werden, welchen Wert die Objekte bekommen sollen, die bisher den zu löschenden Wert nutzen. Mit image:adminhandbuch/button_anwenden.png[Zuweisung anwenden] wird die neue Zuweisung durchgeführt und der Wert gelöscht.
